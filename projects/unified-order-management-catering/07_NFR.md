## Non-Functional Requirements к кейсу "Единая система управления заказами кейтеринга"

### NFR-01 Безопасность
- Все персональные данные должны храниться в зашифрованном виде.
- Передача данных должна осуществляться по HTTPS.
- Доступ к панели управления - только для авторизованных пользователей.
- Пароли должны храниться в хешированном виде.
- Система должна соответствовать требованиям законодательства (уточнить на discovery) о персональных данных.

### NFR-02 Производительность
- Время отклика системы - p95 ≤ 1.5 s.
- Генерация сметы - p95 ≤ 10 s.
- Обновление календаря в реальном времени - p95 ≤ 1 s.
- Система должна поддерживать одновременную работу минимум 100 пользователей без деградации производительности.

### NFR-03 Надёжность
- Система должна поддерживать транзакционность бронирования слотов.
- При сбое оплаты заказ не должен дублироваться.
- Данные заказы должны сохраняться автоматически.
- Система должна иметь механизм логирования ошибок.
- Время доступности системы - не менее 99% в месяц.
- RPO ≤ 15 минут.
- RTO ≤ 2 часа.

### NFR-04 Масштабируемость
- Архитектура системы должна поддерживать горизонтальное масштабирование.
- Система должна выдерживать рост количества заказов минимум в 5 раз без полной переработки архитектуры.

### NFR-05 Юзабилити
- Интерфейс должен быть адаптивным (desktop + mobile).
- Форма заказа должна содержать валидацию полей.
- Ошибки должны отображаться понятным пользователю языком.
- Менеджер должен иметь возможность сортировать и фильтровать заказы.

### NFR-06 Интеграции
- Система должна корректно обрабатывать webhook от платёжного шлюза.
- Webhook/callback обработка должна быть идемпотентной: повторный callback не должен создавать дублей заказов/платежей.
- Время обработки callback - p95 ≤ 3 s.
- Повторная отправка webhook не должна приводить к дублированию операции.

### NFR-07 Хранение и аудит
- История изменений заказа должна храниться не менее 1 года.
- Версии смет должны храниться с возможностью восстановления.
- Резервное копирование базы данных должно выполняться не реже 1 раза в сутки.
