# Кейс: Единая система управления заказами кейтеринга
 
**Автор:** Бизнес-аналитик  
**Заказчик:** Кейтеринговая компания (малый бизнес)

## Описание проекта
Проект направлен на создание единой системы для приёма и обработки заказов кейтеринга с панелью управления и интеграцией с сайтом в реальном времени.  

**Цель:** сократить ручную работу, централизовать данные, исключить овербукинг и ускорить обработку заказов.  
**Целевая аудитория:** клиенты кейтеринга, менеджеры, директор.

## Основные результаты
**MVP:**  
- Панель управления заказами с календарём и карточкой заказа  
- Автоматизация приёма заявок  
- Генерация смет и отправка клиенту  
- Базовые уведомления и безопасность  

**Full (опционально):**  
- Новый сайт с личным кабинетом  
- Интегрированная система онлайн-оплаты  
- Расширенная аналитика и KPI

## Ключевые эффекты
- Время обработки заявки сокращается в 2 раза  
- Потери заказов и овербукинг сведены к минимуму  
- Менеджеры работают по единому стандарту, меньше рутинной работы
